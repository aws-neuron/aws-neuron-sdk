Profiling tools
===============

:ref:`Neuron Explorer <neuron-explorer-home>` is the recommended tool for
profiling, debugging, and optimizing machine learning workloads on AWS
Trainium and Inferentia. It provides Device Trace, System Trace, Hierarchy,
Source Code, Summary, Database, Tensor, and AI Recommendation viewers from
the CLI, web UI, or VS Code.

The earlier Neuron Profiler (``neuron-profile``) and Neuron Profiler 2.0
(Beta) reached end of support in Neuron 2.29. Their user guides are
preserved for reference under :ref:`archived-neuron-profiler`; migrate to
Neuron Explorer using the :ref:`Neuron Explorer FAQ <neuron-explorer-faq>`.
