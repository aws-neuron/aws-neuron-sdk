Neuron no longer supports JAX 0.7 starting with Neuron 2.30
------------------------------------------------------------

Starting with :ref:`Neuron Release 2.30.0 <neuron-2.30.0-whatsnew>`, Neuron no longer supports JAX version 0.7. We recommend that all customers upgrade to the latest supported JAX version.

Customers currently using JAX 0.7 must upgrade to JAX 0.9.0 or later. For more information on supported versions, see :ref:`latest-neuron-release-artifacts`.
