NxD Inference and NxD Core Inference APIs are now in maintenance mode starting with Neuron 2.30
------------------------------------------------------------------------------------------------

Starting with :ref:`Neuron Release 2.30.0 <neuron-2.30.0-whatsnew>`, NxD Inference library and NxD Core Inference APIs are in maintenance mode. Future releases will address critical security issues only and we will gradually end support.

We are actively investing in an enhanced vLLM Neuron plugin that will not require a separate NxD Inference library. More information about the vLLM Neuron plugin enhancements and migration guidance will be shared in the upcoming release.
