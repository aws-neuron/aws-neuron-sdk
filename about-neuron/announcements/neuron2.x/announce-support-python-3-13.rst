Neuron now supports Python 3.13 starting with Neuron 2.30
----------------------------------------------------------

Starting with :ref:`Neuron Release 2.30.0 <neuron-2.30.0-whatsnew>`, Neuron adds support for Python 3.13. Customers can now use Python 3.13 with all Neuron packages and frameworks.

For a list of supported Python versions by package, see :ref:`latest-neuron-release-artifacts`.
