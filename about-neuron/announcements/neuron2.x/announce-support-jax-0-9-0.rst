Neuron now supports JAX 0.9.0 starting with Neuron 2.30
--------------------------------------------------------

Starting with :ref:`Neuron Release 2.30.0 <neuron-2.30.0-whatsnew>`, Neuron adds support for JAX version 0.9.0. Customers can now use JAX 0.9.0 with JAX Neuron for training and inference workloads.

For more information on supported versions, see :ref:`latest-neuron-release-artifacts`.
