## Known Issues and Limitations

Please update to the latest conda package:

```bash
source activate <conda environment>
conda update mxnet-neuron
```

For example, on Conda DLAMI:
```bash
source activate aws_neuron_tensorflow_p36
conda update mxnet-neuron
```
