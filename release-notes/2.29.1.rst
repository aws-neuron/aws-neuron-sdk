AWS Neuron SDK Release Notes - v2.29.1
=======================================

**Release Date**: May 1, 2026   

Release **2.29.1** of the AWS Neuron SDK includes bug fixes applied to the AWS Neuron SDK v2.29.0. See :doc:`the Neuron SDK v2.29.0 release notes </release-notes/prev/2.29.0>` for the full set of changes that shipped with the 2.29.0 release.

Bug Fixes
----------

**Package updates to support this patch**: Neuron Graph Compiler (``neuronx-cc``) ``2.24.5133.0`` --> ``2.24.8799.0``

Neuron Explorer
~~~~~~~~~~~~~~~

* Fixed Neuron Explorer to display DMA information correctly. Previously, Neuron Explorer did not display the details of DMA analysis in the Device Profile pane correctly. 
* Fixed an issue where Neuron Explorer would not correctly show line numbers for NKI workloads.

DLAMIs and DLCs
~~~~~~~~~~~~~~~

* Neuron DLAMIs and DLCs have been updated with the Neuron 2.29.1 packages.
