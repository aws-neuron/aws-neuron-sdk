===================================
TensorBoard plugin (archived)
===================================

.. warning::

   This content is archived. The **Neuron plugin for TensorBoard** (both the
   Inf1 and Trn1 variants) reached end of support. It is provided here for
   historical reference only and is no longer maintained.

   For current profiling, analysis, and optimization workflows, use
   :ref:`Neuron Explorer <neuron-explorer-home>`. Migration guidance lives in
   the :ref:`Neuron Explorer FAQ <neuron-explorer-faq>`.
